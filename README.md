# EduCore.API

A clean and scalable ASP.NET Core Web API for managing students using a 3-Tier Architecture. The project demonstrates best practices in backend development, including DTO usage, separation of concerns, and RESTful API design.

---

## 🚀 Features

- Full CRUD Operations (Create, Read, Update, Delete)
- 3-Tier Architecture (Presentation, Business, Data Access)
- DTO Pattern for clean data transfer
- RESTful API design
- Proper error handling with status codes

---

## 🏗️ Architecture

- Presentation Layer → API Controllers  
- Business Logic Layer → Application logic  
- Data Access Layer → Data handling  

---

## 🧠 Technologies Used

- ASP.NET Core Web API  
- C#  
- LINQ  
- DTO Pattern  

---

## 📌 API Endpoints

| Method | Endpoint                          | Description              |
|--------|----------------------------------|--------------------------|
| GET    | /api/Students                    | Get all students         |
| GET    | /api/Students/IdStudent/{id}     | Get student by ID        |
| POST   | /api/Students                    | Create new student       |
| PUT    | /api/Students/{id}               | Update student           |
| DELETE | /api/Students/Delete/{id}        | Delete student           |
